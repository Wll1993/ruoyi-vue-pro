## 用法

### 安装 pnpm

```
# 查看当前 npm 源

npm config ls

# 如果执行上面命令您并未看到 registry = "https://registry.npmjs.org/"，说明使用的非npm官方源，请执行下面命令

npm config set registry https://registry.npmjs.org

# 如果您还没安装 pnpm，请执行下面命令

npm install -g pnpm

# mac 用户遇到安装报错请在命令前加上 sudo

# 安装依赖

pnpm install

# 运行项目

pnpm run dev

# 安装一个包

pnpm add 包名

# 卸载一个包

pnpm remove 包名
```
